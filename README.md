# Olist Delivery Time Prediction

This project analyzes customer order delivery times using the Brazilian E-Commerce Public Dataset by Olist.  
The goal is to predict estimated delivery duration (in days) based on order, customer, seller, and product information.

The workflow includes data preprocessing, exploratory data analysis (EDA), feature engineering, and regression modeling.
